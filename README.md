# resolution-tools
Resolution Tool — Shizuku Edition
Aplikasi pengubah resolusi HP Android tanpa root menggunakan Shizuku.
Cara Aktifin Shizuku
1. Install Shizuku
Download di Play Store → cari Shizuku → install
2. Aktifkan Developer Options di HP
Buka Settings → About Phone
Ketuk Build Number sebanyak 7 kali
Masuk ke Settings → Developer Options
3. Aktifkan Wireless Debugging
Di Developer Options → cari Wireless Debugging → ON
Tap Wireless Debugging → tap Pair device with pairing code
Catat IP address, Port, dan Pairing Code yang muncul
4. Aktifkan Shizuku via Wireless Debugging
Buka app Shizuku
Tap Pair using Wireless Debugging
Masukkan IP, Port, dan Pairing Code tadi
Shizuku akan aktif — statusnya jadi hijau
5. Buka Resolution Tool
Pilih resolusi yang diinginkan
Tap Terapkan Resolusi
Muncul dialog peringatan → tap Lanjutkan
Shizuku minta izin → tap Izinkan
Resolusi HP langsung berubah
Reset Resolusi
Tap tombol Reset ke Default kapan saja untuk kembali ke resolusi awal.
Catatan
Shizuku harus diaktifkan ulang setiap kali HP restart
Tidak perlu root
Semua risiko ditanggung pengguna sendiri
